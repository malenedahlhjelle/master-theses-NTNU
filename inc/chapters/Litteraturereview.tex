\subsubsection{Traditional methods}
Traditional path planning methods can be divided into graph-based path planning and sampling based path planning. Graph based path planning is based on constructing a graph on a formerly known map and using an algortim like Djikstra or A* to find the shortest path between points in the map. Sampling based path planning is based on iteratively expanding the graph to connect two points in the map.Both methods are based on having a predefined static map of the environment.
